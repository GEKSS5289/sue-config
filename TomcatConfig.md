# 🐱Tomcat9.0.33

- 🚬下载
- 📦解压
- ⚗测试
# 🚬到apache官网下载Tomcat9.0.33
   📞[tomcat9.0.33](https://archive.apache.org/dist/tomcat/tomcat-9)
# 📦解压

# ⚗测试
    cd /apache-9.0.33/bin
    ./startup.sh
